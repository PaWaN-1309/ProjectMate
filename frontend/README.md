# ProjectMate (frontend)

Minimal frontend for ProjectMate (React + Vite + Tailwind).

Setup

1. Install deps

   npm install

2. Run dev server

   npm run dev

Notes
- Tailwind is configured via `tailwind.config.cjs` and `postcss.config.cjs`.
- The frontend currently uses a small mock API client in `src/api/client.js`. Replace with real endpoints when backend is ready.
